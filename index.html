Observed Meteorological Fields Compared to Self-Organized Maps

This page will display comparisons of observed meterological fields and observed precipitation fields to the Self-Organized Maps that they have been assigned to. I will start out with some of the wettest 6 hour periods observed in 2022. Upper level meteorological fields were generated in Python using data from the NCAR Reanalysis dataset. Surface precipitation images were also generated in Python, using data collected from the National Weather Service (NWS) Real-Time Mesoscale Analysis (RTMA) database.
